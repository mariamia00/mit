Această tabără de vară se concentrează pe fundamentele dezvoltării paginilor web folosind HTML și CSS. Participanții vor învăța:

  • Structurarea paginilor web folosind HTML.
  • Stilizarea și design-ul paginilor web cu CSS si SASS. Animații în CSS.
  • Principii de design responsive pentru a asigura compatibilitatea pe diverse dispozitive.
  • Introducere în instrumentele moderne de dezvoltare web, cum ar fi Flexbox și Grid Layout.
  • Introducere în framework-uri CSS, cum ar fi Bootstrap și Tailwinds

Tabără destinată celor care doresc să își aprofundeze cunoștințele în JavaScript, limbajul de programare esențial pentru dezvoltarea web interactivă. Subiectele includ:

  • Bazele JavaScript: variabile, funcții, bucle și condiționale
  • Manipularea DOM-ului pentru a crea pagini web interactive
  • Utilizarea API-urilor browser-ului pentru diverse funcționalități
  • Introducere în ES6 și alte caracteristici moderne ale JavaScript
  • Asincronizare în JavaScript: Promises, async/await
  • Programare funcțională și funcții de ordine superioară
  • Optimizare a performanței și debugging avansat
  • Crearea și gestionarea obiectelor în JavaScript

O tabără este ideală pentru cei care vor să învețe să construiască aplicații web folosind Angular, un framework puternic dezvoltat de Google. Programul include:

  • Introducere în structura proiectelor în Angular. Diferențe între vechiul și noul Angular
  • Introducere în TypeScript, limbajul folosit de Angular
  • Conectarea dinamică între date și interfața utilizatorului
  • Structurarea aplicațiilor folosind componente Angular și rutare
  • Servicii, dependențe și asincronizare cu observabile
  • Utilizarea formularelor reactive și template-driven

Programare C++

În acest curs vei învăța programare în C++, un limbaj de bază pentru dezvoltarea de jocuri și aplicații. C++ este folosit în dezvoltarea de software performant și jocuri video, oferindu-ți o fundație solidă în programare.

Ce vei învăța:

  • Fundamentele C++: Sintaxă, structuri de date și concepte esențiale.
  • Programare orientată pe obiecte: Clase și obiecte.
  • Dezvoltare de jocuri: Crearea de jocuri simple și utilizarea bibliotecilor grafice.

De ce să alegi acest curs:

  • Versatilitate și performanță: C++ este ideal pentru aplicații de înaltă performanță și dezvoltarea de jocuri.
  • Abilități valoroase: Înțelegerea C++ îți va oferi abilități apreciate pe piața muncii și te va pregăti pentru dezvoltarea de software complex.
  • Proiecte practice: Vei avea ocazia să lucrezi la proiecte reale care îți vor îmbunătăți portofoliul și îți vor oferi experiență practică.
  • Comunitate și resurse: Beneficiezi de o comunitate activă de dezvoltatori și o gamă variată de resurse online pentru suport și învățare continuă.
Web Developement

Acest curs te va învăța cum să creezi un site web complet de la zero, folosind tehnologiile fundamentale ale webului: HTML, CSS și JavaScript. Vei învăța să dezvolți site-uri interactive și funcționale și să înțelegi cum funcționează diferitele componente ale unei aplicații web moderne.

Ce vei învăța:

  • HTML și CSS: Crearea structurii și stilizarea paginilor web, de la elementele de bază la tehnici avansate.
  • JavaScript: Programare interactivă pe web, inclusiv crearea unui mic joc și înțelegerea DOM-ului.
  • MySQL: Noțiuni de bază despre baze de date, cum să creezi și să gestionezi datele utilizatorilor.
  • Angular: Introducere în Angular, un framework care facilitează dezvoltarea web prin structurarea codului HTML, CSS și TypeScript pentru performanță și ușurință în utilizare.

De ce să alegi acest curs:

  • Învățare completă: Acest curs acoperă toate aspectele esențiale ale dezvoltării web, de la front-end la back-end.
  • Proiecte practice: Vei lucra la proiecte reale, inclusiv crearea unui site complet și a unui joc mic, pentru a-ți dezvolta abilități practice.
  • Tehnologii avansate: Vei învăța cum să folosești Angular pentru a crea aplicații web performante și bine structurate.
  • Certificare valoroasă: Abilitățile în HTML, CSS, JavaScript, MySQL și Angular sunt foarte căutate pe piața muncii.
Taberele de vară

Intră în fascinanta lume a graficii 3D! Acest curs interactiv este destinat elevilor de liceu pasionați de jocuri video și tehnologia de imprimare 3D. Vei învăța tehnici esențiale de modelare, texturare și animație, folosind software-uri populare din industrie, precum și metode de optimizare pentru imprimarea 3D.

Ce vei învăța:

  • Grafică 3D pentru jocuri: Crearea de modele 3D, aplicarea texturilor și animațiilor pentru a aduce la viață proiectele tale.
  • Imprimare 3D: Pregătirea și optimizarea modelelor pentru imprimare, alături de tehnici de finisare pentru îmbunătățirea aspectului și funcționalității.
  • Portofoliu și carieră: Sfaturi despre cum să îți construiești un portofoliu profesional și să explorezi oportunitățile de angajare sau freelancing în domeniul graficii 3D.

De ce să alegi acest curs:

  • Învățare practică: Vei lucra la proiecte reale, dezvoltând modele și animații pentru jocuri sau obiecte pentru imprimare 3D.
  • Tehnici avansate: Vei învăța să folosești instrumente esențiale din industrie, dezvoltându-ți abilitățile creative și tehnice.
  • Pregătire pentru viitor: Acest curs îți oferă cunoștințele necesare pentru a începe o carieră în grafică 3D sau imprimare 3D.
